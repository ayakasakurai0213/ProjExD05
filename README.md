# ゲーム のタイトル
* 「RPG of くそげー」
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* RPGの戦闘シーン

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
