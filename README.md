# ゲーム のタイトル
* 「RPG of くそげー」
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* RPGの戦闘シーン
* 敵のスライムと戦闘をする
* 攻撃をクリック: スライムに攻撃
* 防御をクリック: 防御
* 回復をクリック: HPを回復
* 調教をクリック: スライムが仲間になる
* HPが0になると死亡、ゲームオーバー
